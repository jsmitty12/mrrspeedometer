Uploading to Firebase
---------------------

1. `yarn build`
2. `yarn deploy`


### Initial Firebase Setup

1. Authenticate
    `yarn firebase login`
2. List available projects
    `yarn firebase list`
3. Set the active project
    `yarn firebase use <project-name>`
